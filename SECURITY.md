# Security Policy

**Do not open a public GitHub issue for security vulnerabilities.**

If you believe you have found a security vulnerability in ARA Community Edition, please email **[security@aralabs.ai](mailto:security@aralabs.ai)** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Your ARA version and platform

We will acknowledge receipt within **48 hours** and aim to resolve critical issues within **7 days**.

See [aralabs.ai/security](https://aralabs.ai/security) for more information on our security posture and responsible disclosure policy.
